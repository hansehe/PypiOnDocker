# Pypi On Docker

Deploy your personal pypi register with Docker!
Basically, follow this installation: [https://hub.docker.com/r/mosquito/pypi-server/](https://hub.docker.com/r/mosquito/pypi-server/)

## Requirements:
- Docker
- Python + pip
- `pip install SwarmManagement`

## Run & Stop
- `swm -start`
- `swm -stop`
